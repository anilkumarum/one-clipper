How to use oneClipper

Pin oneClipper after install extension
Click on oneClipper or (Alt + N ) to open oneClipper popup Window

    Action buttions
  1. Bookmark: Bookmark current tab in notion
  2. Screenshot: Capture screenshot and insert into notion (full screenshot coming soon)
  3. Clip Article: oneClipper auto clipped current page
  4. Multi Select text:click on paragraph which you want to clip then click on "Clip to notion" button on right corner
Format Action Keyboard Shortcut
Screenshot Alt+S
Clip Article Alt+A
Multi Select text Alt+M

Take screenshot

Click on screenshot button for capture and save screenshot
Use Keyboard shortcut (Alt + S)

Clip Article

Click on oneClipper icon to open extension's popup window.
Keyboard shortcut: Alt+A(when popup window open)
Clip Article button to clip article.
Clipped article preview dialog popup
Select notebook's section where you want to save this article
Click on Clip article button to save article. oneClipper save clipped article in selected database with url,tags

Append selected text

You can append text in 3 ways

  1. Mutli select text
    Click on Mutli Select text from popup window
    Keyboard shortcut: Alt+M(when popup window open)
    Click on paragraph which you want to clip.
    When you done, click on Clip selected button on right bottom corner
  2. Right click menu
    Select text which you want to clip then
    hover Clip to notion option from right click menu
    then select page where you want to save text
    Selected text will highlighted and insert into selected pages
  3. Side panel Ctrl+C copy action
    Click on open side panel to open oneClipper sidePanel
    Open page where you want to save text
    Select text from any webpage then press Ctrl+C
    copied text will auto-inserted into selected page
    Ctrl+Z for undo
    Ctrl+Y for redo
  4. Drag-n-drop
    Click on Select text to open oneClipper sidePanel
    Click on Open pages to open existing page.
    Drag-n-drop selected text or images.
    Click on Insert button to append text in local page file.
    Ctrl+Z for undo
    Ctrl+Y for redo
    Shift+Space for extend selection
    Alt+H for highlight text

Highlight text

When you clip selected text, selected text will highlighted automatically
Click on Save Higlight button on right bottom corner to save highlight
For disable highlight, Go to options page

Insert image

  1. Right click menu
    Right click over image which you want to save
    Hover Clip to notion option from right click menu
    then select page where you want to save image
    Selected image will saved in current vault and add image path in selected page file
  2. Drag-n-drop
    Click on Select text to open oneClipper sidePanel
    Click on Open pages to open existing page.
    Drag-n-drop images.
    Drop images will uploaded to notion

Clip text,urls and images by right click menu

Right click on selected content then hover over "clip to notion" option
Select page from sub-menu list and selected content will insert into notion
